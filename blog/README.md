# 小宝成长日记静态站点

此目录包含记录小宝成长的静态博客页面，包含文字、图片与视频展示模块。当前提供了一组自动生成的占位图片与 5 秒彩色背景视频，可先用于预览布局；正式发布前，请将真实素材放入 `assets/images` 与 `assets/videos` 中，并保持文件命名与页面引用一致。

## 文件结构
- `index.html`：首页，展示里程碑与站点导航。
- `memories.html`：成长瞬间，记录图文日记。
- `gallery.html`：照片墙，以图集呈现精彩瞬间。
- `videos.html`：影像集，嵌入家庭视频。
- `privacy.html`：隐私政策说明。
- `styles/main.css`：全站共享样式。
- `assets/images`：放置 `.jpg`、`.png` 等图片资源。
- `assets/videos`：放置 `.mp4` 等视频资源。

## 使用方式
1. 在浏览器直接打开任一 HTML 文件即可预览。
2. 使用静态服务器（如 `npx serve`）可获得更完整的链接导航体验。
3. 更新素材后，请同步校对页面上 `alt` 描述与文案，确保可访问性。
